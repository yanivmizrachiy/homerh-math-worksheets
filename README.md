# מערכת יצירת דפי עבודה במתמטיקה - כיתה ח'

מערכת מקצועית ומדויקת ליצירת דפי עבודה במתמטיקה בעברית, מוכנים להדפסה על דף A4, בהתאם לפרומפט EXTREME.

## מבנה הפרויקט

```
homerh/
├── .cursorrules              # הוראות מערכת ל-AI
├── EXTREME_PROMPT.md        # הפרומפט המחייב ליצירת דפי עבודה
├── requirements.txt         # תלויות Python
├── README.md               # קובץ זה
│
├── core/                   # ליבת המערכת
│   ├── graph_engine.py    # מנוע יצירת גרפים מקצועיים
│   ├── pdf_engine.py      # מנוע יצירת PDF ל-A4
│   └── worksheet_validator.py  # בודק איכות אוטומטי
│
├── worksheets/             # דפי עבודה מוכנים
│   └── grade-8/           # כיתה ח'
│       ├── kavba_a1_graph_reading.md
│       ├── kavba_a1_slope_table.md
│       └── kavba_a1_50_questions_coefficients.md
│
└── assets/                 # נכסים (גרפים, תמונות)
    └── graphs/
        └── jerusalem_motion_graph.png
```

## התקנה

```bash
pip install -r requirements.txt
```

## שימוש

### יצירת גרף

```bash
python core/graph_engine.py
```

יוצר גרף תנועה ירושלים ב-`assets/graphs/jerusalem_motion_graph.png`

### יצירת PDF מדף עבודה

```bash
python core/pdf_engine.py worksheets/grade-8/kavba_a1_graph_reading.md
```

יוצר קובץ PDF מוכן להדפסה: `worksheets/grade-8/kavba_a1_graph_reading.pdf`

### בדיקת איכות דף עבודה

```bash
python core/worksheet_validator.py worksheets/grade-8/kavba_a1_graph_reading.md
```

מריץ בדיקות איכות מחמירות על דף העבודה.

## דפי עבודה מוכנים

### 1. קריאת גרף תנועה
**קובץ**: `worksheets/grade-8/kavba_a1_graph_reading.md`

גרף תנועה: מרחק מהבית בירושלים (0-6 שעות, 0-12 ק"מ)
- 7 נקודות מסומנות: A(0,0), B(1,4), C(2,4), D(3,8), E(4,6), F(5,6), G(6,0)
- 6 סעיפים: (א) קריאה ישירה, (ב) מציאת זמן, (ג) משמעות נקודה, (ד) משמעות קטע, (ה) השוואה, (ו) אוריינות מתמטית

### 2. משמעות השיפוע דרך טבלה
**קובץ**: `worksheets/grade-8/kavba_a1_slope_table.md`

טבלת ערכים X, Y - הבנת השיפוע ללא נוסחאות
- 6 סעיפים: (א) זיהוי שינוי, (ב) השלמת טבלה, (ג-ד) משפטי השלמה, (ה) קשר לגרף, (ו) אוריינות

### 3. 50 שאלות על מקדמים
**קובץ**: `worksheets/grade-8/kavba_a1_50_questions_coefficients.md`

50 שאלות מגוונות בסיפורים יומיומיים על משמעות המקדמים בפונקציה קווית
- מגוון סוגי שאלות: זיהוי מקדם, הבנת משמעות, השוואות, ועוד

## מפרט טכני

### גרפים
- DPI: 400 (מינימום)
- Figsize: 18×12 אינץ'
- פונטים: מינימום 24pt לטקסט, 40pt לתוויות
- קווים: מינימום 4px עובי
- נקודות: רדיוס 0.25 יחידות

### PDF
- A4: 210×297mm
- שוליים: Top/Bottom 25mm, Sides 20mm
- פונט: Heebo/Assistant/David Libre
- RTL מושלם
- LaTeX MathJax

### דפי עבודה
- כותרת ברורה
- הוראות קצרות וידידותיות
- מספיק מקום לכתיבה (3-5 שורות לכל פתרון)
- עיצוב אחיד ומקצועי
- ללא פתרונות או רמזים

## עקרונות איכות

1. **קוד ראשון, תצוגה אחרי** - כל גרף וחישוב מאומתים בקוד
2. **בדיקות אוטומטיות** - כל קובץ נבדק לפני סיום
3. **פרומפט EXTREME הוא חוק** - אין סטיות
4. **כיתה ח' בלבד** - אין חומר מתקדם
5. **A4 הדפסה בלבד** - אין עיצוב למסך
6. **עברית מושלמת** - RTL, טרמינולוגיה, תקינות
